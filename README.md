# Редактирование Электронного дневника.

Этот скрипт (файл под названием 'scripts.py' в данном репозитории) содержит функции,  которые помогут ученику отредактировать данные в базе данных электронных дневников учеников. А именно:
1. Удаление замечаний от учителя (функция 'delete_chastisements')
2. Создание записи с похвалой учителя привязанные к рандомным датам с рандомным текстом (функция 'create_commendation')
3. Исправляет оценки 3 и 2 на 5 (функция 'fix_marks')

## Окружение

Python3.14.2 должен быть уже установлен.
Рекомендуется использовать [virtualemv/venv](https://docs.python.org/3/library/venv.html) для изоляции проекта.

## Запуск

Для того чтобы запустить данный скрипт, необходимо:
1. Поместить его в каталог [этого](https://github.com/Vitaliusius/django3-lesson) репозитория с файлом 'manage.py'. 
2. Запустите сервер командой ```python3 manage.py runserver``` из консоли (подробнее в файле README.md  в репозитории из пункта номер 1)
3. Открыть новое окно консоли и запустить Django Shell с помощью команды ```python manage.py shell```
4. Затем импортировать все модели из файла 'models.py' чтобы к ним был доступ с помощью команд:
```
from datacenter.models import Schoolkid
from datacenter.models import Teacher
from datacenter.models import Subject
from datacenter.models import Lesson
from datacenter.models import Mark
from datacenter.models import Chastisement
from datacenter.models import Commendation
```

Затем импортиируем нужные для работы функций объекты:
 
```
from django.core.exceptions import ObjectDoesNotExist
import random
```

5. Далее выбираем нужную функцию и импортируем ее из файла 'scripts.py'  комнандой ```from scripts import <имя_функции>```
6. Все функции приниают в виде аргумента имя как отдельно, так и с фамилией или отчеством ученика, или полностью ФИО написанных с заглавной буквы и через пробел. Но функция 'create_commendation' принимает так же в виде второго аргумента название предмета написанного с заглавной буквы. В случае если имя не существует в базе данных в консоль выведется сообщение 'Нет такого имени', а если в базах несколько учеников с указанным именем выведется сообщение 'Нашлось несколько имен'. Поэтому желательно вводить хотя бы имя и фамилию. 
Пример команд для вызова функций:

```
	create_commendation('Иван Фролов', 'Музыка')
	delete_chastisements('Иван Фролов')
	fix_marks('Иван Фролов')
```
